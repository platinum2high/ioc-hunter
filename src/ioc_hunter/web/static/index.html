What this is

  • Async TI engine. Seven feeds in parallel: Tor exit, NetMeta (bogon/private), URLhaus, ThreatFox, AbuseIPDB, OTX, VirusTotal.
  • Transparent scoring. Weighted by source confidence — no black box.
  • Defang-aware. Paste evil[.]com, hxxp://, [at] — refanged automatically.
  • Same code as the CLI. This page is a thin layer over the open-source engine. pip install ioc-hunter for the full thing (parse-eml, watch-mode, STIX/MISP/Sigma export, correlator).
